Solution: Reset xcode-select and install command line tools.

Step 1: Reset xcode-select

(Enter your Mac password when asked)

Step 2: Install command line tools

What happens:
- Dialog appears
- Click "Install"
- Wait 5-10 minutes
- Installs ~1GB (NOT full Xcode)

Step 3: Verify it worked
xcode-select -p

Should show: /Library/Developer/CommandLineTools
(If it shows /Applications/Xcode.app, installation didn't work)
